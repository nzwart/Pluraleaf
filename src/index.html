Here's the index page for Pluraleaf's database interface.

  • The Categories page contains a table of all the current possible categories a Pluraleaf pin can have. You can create a category on that page.
  • The Pins page has a table of all pins saved by all users. You can create a pin on that page.
  • The Saved Pins page has a table of all pins that have been saved by all users. On that page, you can create a new save, as well as update the date of a previous save, and lastly delete a save altogether, without deleting the user or original pin associated with that save.
  • The Users page has a table of all registered users. You can create a user on that page.
  • The Comments page has a table of all comments on all pins. You can create a comment on that page.
